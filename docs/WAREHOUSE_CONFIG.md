# Конфиг склада

> Этот файл заменён полной документацией системы склада: **[`WAREHOUSE.md`](./WAREHOUSE.md)**.
>
> Там описаны все флаги конфига, формат `items.json` и `crates/`, пулы очков,
> NPC-кладовщик, зона приёма, личный бюджет, сдача снаряжения, команды и права.

Краткие пути:

- Каталог предметов: `config/pjmbasemod/warehouse/items.json`
- Ящики поставок: `config/pjmbasemod/warehouse/crates/`
- Перезагрузка без рестарта: `/pjm config reload warehouse`
